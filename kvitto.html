Google-inställningar
Klistra in ditt Google OAuth Client ID. Du skapar det på console.cloud.google.com under APIs & Services → Google Auth Platform → Clients.

1Aktivera Gmail API

APIs & Services → Library → sök "Gmail API" → Aktivera.

2Google Auth Platform

APIs & Services → Google Auth Platform → Branding (appnamn "Kvitto") → Audience (External, lägg till din Gmail som testanvändare).

3Skapa Client ID

Clients → Create client → Web application → Authorized JavaScript origins: https://Robdancer666.github.io
